## Overview

The Web UI provides a complete web interface for Agency Swarm, porting all terminal features to a modern web application. Built with Next.js, React, TypeScript, and FastAPI.

<Features>
  <Feature icon="comment" title="Real-time Streaming">
    Server-Sent Events (SSE) for instant response streaming
  </Feature>
  <Feature icon="users" title="Multi-Agent">
    Agent selector with @agent mention support
  </Feature>
  <Feature icon="terminal" title="All Commands">
    All slash commands from terminal: /help, /new, /compact, /resume, /status, /cost
  </Feature>
  <Feature icon="floppy-disk" title="Persistence">
    Chat history with sidebar and auto-save
  </Feature>
  <Feature icon="chart-line" title="Usage Tracking">
    Token count and cost visualization panel
  </Feature>
  <Feature icon="keyboard" title="Keyboard Shortcuts">
    Ctrl+K for commands, Ctrl+/ for focus, ESC to cancel
  </Feature>
</Features>

## Quick Start

<Accordion title="Step 1: Install Frontend Dependencies" defaultOpen={false}>

Navigate to the web UI directory and install Node.js dependencies:

```bash
cd agency-swarm/src/agency_swarm/ui/demos/web
npm install
```

<Note>Requires Node.js 18+ and npm or yarn.</Note>

</Accordion>

<Accordion title="Step 2: Install Backend Dependencies" defaultOpen={false}>

Install Python dependencies for the FastAPI backend:

```bash
cd backend
pip install -r requirements.txt
```

</Accordion>

<Accordion title="Step 3: Configure Your Agency" defaultOpen={true}>

Edit `backend/main.py` to configure your agents:

```python
from agency_swarm import Agency, Agent

def create_agency() -> Agency:
    ceo = Agent(
        name="CEO",
        description="Agency coordinator",
        instructions="You are the CEO. Coordinate tasks and delegate.",
        model="gpt-4o-mini",
    )

    developer = Agent(
        name="Developer",
        description="Code and technical tasks",
        instructions="You are a Developer. Write code and solve problems.",
        model="gpt-4o-mini",
    )

    return Agency(
        agents=[ceo, developer],
        default_agent=ceo,
        shared_instructions="You are part of an AI agency.",
    )
```

</Accordion>

<Accordion title="Step 4: Run the Backend" defaultOpen={false}>

Start the FastAPI server:

```bash
cd backend
python main.py
```

The backend will run on `http://localhost:8000`

</Accordion>

<Accordion title="Step 5: Run the Frontend" defaultOpen={false}>

In a new terminal, start the Next.js development server:

```bash
npm run dev
```

The UI will be available at `http://localhost:3000`

</Accordion>

## Features

### Real-Time Streaming

The Web UI uses Server-Sent Events (SSE) to stream responses in real-time, just like the terminal interface.

### Multi-Agent Communication

- **Agent Selector**: Dropdown to choose which agent to message
- **@agent Syntax**: Type `@agentname` in the chat to mention a specific agent
- **Handoffs**: Agents can transfer conversations to each other

### Slash Commands

All terminal commands are available:

| Command | Description |
|---------|-------------|
| `/help` | Show available commands |
| `/new` | Start a new chat session |
| `/compact [instructions]` | Summarize and continue conversation |
| `/resume [chat_id]` | Resume a previous conversation |
| `/status` | Show agency status and setup |
| `/cost` | Display usage and costs |

Press `Ctrl+K` to open the command palette.

### Chat Persistence

- **Auto-save**: Chats are saved after each message
- **Sidebar**: View all saved conversations
- **Metadata**: Each chat has summary, timestamps, and usage info
- **Resume**: Easily continue previous conversations

### Usage Tracking

Press the cost button (or use `/cost`) to see:

- Total tokens (input, output, cached)
- Request count
- Total cost in USD
- Token distribution chart

## Environment Variables

Create a `.env.local` file in the web directory:

```bash
# Python Backend URL
PYTHON_BACKEND_URL=http://localhost:8000

# Optional: Custom chats directory
# AGENCY_SWARM_CHATS_DIR=./chats
```
